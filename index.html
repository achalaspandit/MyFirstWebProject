1. Bin-Py

The aim is to simplify garbage collection process of workers from Public Areas It is a challenging and painful process for workers to segregate waste. It is always better to segregate waste as disposing. However in case of public areas there is no supervisor to manage this. Our prototype is a smart bin that seperates various waste to different bins. Using the waste classification dataset from kaggle we have a trained a model using Convolution Neural Network over a huge dataset of image samples. We further are planning to extend the project by adding sensor components, to have a realtime update of status of bins.
The code to this project is available on github

2. Easy Tax

This Project is designed to ease out the process of Tax Calculation. Various inputs are taken from the user regarding personal information and their past payment history of payments, donations and so on. Finally calculations are made to find the remaining tax amount that is due for the respective assesment year.
